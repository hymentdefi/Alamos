# Competidores

## TL;DR

Mercado fragmentado. Ningún player ofrece los 3 mercados (AR, US, crypto) con conversión nativa y UX para primer inversor.

## Fintechs con inversiones

| Competidor | Fortaleza | Debilidad |
|---|---|---|
| Ualá | Base masiva, marca fuerte | Inversiones es secondary feature |
| Lemon Cash | Fuerte crypto, UX joven | Expuesto regulatoriamente, poco diversificado |
| Belo | Crypto + USD, buena UX | Chico, sin mercado AR tradicional |
| Buenbit | Early mover crypto | Problemas regulatorios, perdió momentum |

## Brokers / ALyCs

| Competidor | Fortaleza | Debilidad |
|---|---|---|
| Cocos Capital | UI moderna, comisiones bajas | Sigue siendo broker, no retail-first |
| IOL invertironline | Más grande, más productos | UX anticuada, orientado a trader |
| Bull Market Brokers | Amplio catálogo | UX pobre |

## Nuestra diferenciación

1. Single-app multi-mercado: AR + US + crypto en una experiencia
2. Conversión nativa ARS ↔ USD ↔ USDT como feature core
3. Diseñado para primer inversor, no para traders
